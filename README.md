# AutoRia Car Scraper

Цей проект — автоматизований інструмент для збору (парсингу) даних про вживані автомобілі з сайту [Auto.ria.com](https://auto.ria.com/).

Застосунок імітує поведінку реального користувача, обходить базові захисти (банери куків, приховані телефони), збирає детальну інформацію про авто та зберігає її у базу даних **PostgreSQL**. Проєкт повністю контейнеризований за допомогою **Docker**.

## Основні можливості

* **Глибокий парсинг:** Збирає заголовок, ціну ($ та грн), пробіг, VIN-код, держ. номер, ім'я продавця, посилання на фото.
* **Витягування телефону:** Вміє натискати кнопку "Показати телефон", обходити "липкі" банери та хедери, і діставати реальний номер продавця.
* **Розумна пагінація:** Проходить по сторінках пошуку до закінчення оголошень.
* **Стійкість (Robustness):** Вбудований механізм `wait_for_db` для очікування готовності бази даних та обробка помилок під час парсингу.
* **Планувальник (Scheduler):** Автоматичний запуск парсингу щодня у заданий час.
* **Автоматичні бекапи:** Створення SQL-дампів бази даних щодня.
* **Docker:** Легкий запуск однією командою.

## Технологічний стек

* **Python 3.11**
* **Selenium WebDriver** (Chrome) — для емуляції браузера та взаємодії з JS.
* **BeautifulSoup4** — для швидкого парсингу HTML.
* **SQLAlchemy (ORM)** — для роботи з базою даних.
* **PostgreSQL** — основна база даних.
* **Docker & Docker Compose** — для контейнеризації.
* **Schedule** — для планування задач.

## Як запустити проєкт

### 1. Попередні вимоги
На комп'ютері має бути встановлено:
* [Docker](https://www.docker.com/get-started)
* Docker Compose

### 2. Клонування репозиторію
Відкрийте термінал і виконайте команди:

```bash
git clone <https://github.com/kkkkkkkkatya/auto_ria_scraper.git>
cd auto_ria_scraper
```

Створіть файл .env у кореневій папці проєкту і вставте туди наступні налаштування:
POSTGRES_USER=postgres
POSTGRES_PASSWORD=mysecretpassword
POSTGRES_DB=autoria_db
DB_HOST=db
DB_PORT=5432
SCRAPE_TIME=12:00

Зберіть та запустіть контейнери однією командою:
```bash
docker-compose up --build
```
